# Product Overview

ShieldGate is a comprehensive OAuth 2.0 and OpenID Connect Authorization Server built with Go. It provides centralized authentication and authorization services for modern applications.

## Core Purpose
- **OAuth 2.0 Compliance**: Full support for OAuth 2.0 specification (RFC 6749)
- **OpenID Connect**: Complete OIDC implementation for identity management
- **Security-First**: Enhanced security with PKCE, JWT tokens, and comprehensive validation

## Supported Features
- **Grant Types**: Authorization Code (with PKCE), Client Credentials, Refresh Token
- **Client Types**: Traditional web apps, mobile apps, Single Page Applications (SPAs)
- **Security**: HTTPS/TLS mandatory, bcrypt password hashing, rate limiting
- **Standards Compliance**: OAuth 2.0, OpenID Connect, JWT tokens
- **Performance**: Redis caching support, optimized database queries

## Target Use Cases
- Microservice authentication and authorization
- Single Sign-On (SSO) for enterprise applications
- API access control and token management
- Identity provider for third-party integrations
- Secure client credential management

## Key Differentiators
- Production-ready with comprehensive security features
- Docker-ready deployment with PostgreSQL and Redis
- Extensive logging and monitoring capabilities
- Vietnamese and English documentation support
- Modular architecture for easy customization
